Uso de menejo de GIT

La versión del git

git --version

Archivo oculto de git

git init

Ver el estado del git

git status

Adiccione los archivos individual

git add "nombre del archivo"

Adiccione varios archivos

git add *

Ver el estado del git si hay modificaciones en los archivos

git status

Para renombrar el archivo en el git

git commit -m "nombre"

Crear un repositorio en git hub

Crear repositorio

El origen del git

git remote add origin

Verificar la conexión

git remote -v

Subir el archivo en el repositorio

git push -u origin master
